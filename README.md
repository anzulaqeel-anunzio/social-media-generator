# Awesome Social Image

Generate beautiful Open Graph (social preview) images for your Awesome Lists automatically.

## Features

-   **Automatic Extraction**: Reads title and description from your `README.md`.
-   **Customizable**: Choose colors and output filenames.
-   **High Quality**: Generates a crisp PNG image suitable for Twitter/X cards and link previews.

## Installation

```bash
npm install -g awesome-social-image
```

## Usage

Generate an image from the current directory's `README.md`:

```bash
awesome-social-image
```

Options:
- `-i, --input <file>`: Input Markdown file (default: "README.md")
- `-o, --output <file>`: Output image file (default: "og-image.png")
- `-c, --color <hex>`: Accent color for the banner (default: "#0969da")

Example:

```bash
awesome-social-image -i README.md -o preview.png -c #ff0000
```
